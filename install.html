Инструкция по установке

1. Разархивируйте архив estate_rel.zip в корне вашего web-сайта, либо в подкаталог, например /dom/ (не используйте /realty/ - это слово зарезервировано)

2. Зайдите на этот сайт

3. Будет запущен инсталлер, следуйте инструкциям

------------

Если вы не знакомы с установкой PHP-скриптов, то можете попробовать скрипт на нашем хостинге.

По этой ссылке: http://www.sitebill.ru/client/cart.php?a=add&pid=3

Для вас будет создан сайт с доступом в панель управления. На почтовый ящик, указанный при регистрации будут отправлены логин и пароль.

------------

Также свежая версия инструкции с видео всегда есть на сайте http://www.sitebill.ru/quick-start/
