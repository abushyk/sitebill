Шаблон Realia

1. Использует верисю jQuery не своместимую со штатным WYSIWYG-редактором CLEditor. 
Для корректной работы шаблона редактором для полей типа textarea_editor в Настройках - Общее - WYSIWYG-редактор должен быть установлен CKEditor.
Папка с редактором должна находиться в корне Вашего сайта и иметь имя ckeditor.
Если этой папки нет, редактор необходимо скачать (http://ckeditor.com/download) и поместить в эту папку.

2. Шаблон интегрирует возможности приложения Клиент. Приложение должно быть установлено и активно. Минимальная версия приложения 1.0.2
Так же необходимо наличие модели contactus для приемки сообщений от пользователя.
Если эти возможности Вам не нужны, обратитесь в техподдержку для модификации Вашего шаблона.

3. Шаблон не поддерживает ветвистых структур. Избегайте сильной вложенности пунктов структуры и расположения особо ветвистых разделов на правом краю навигации.
